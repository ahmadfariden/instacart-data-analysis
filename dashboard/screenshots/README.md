# Dashboard Screenshots

Taruh screenshot 4 halaman dashboard Power BI kamu di folder ini dengan nama file:

- `01_overview.png`
- `02_time_pattern.png`
- `03_product_basket.png`
- `04_customer_segmentation.png`

Cara ambil screenshot yang rapi:
1. Buka Power BI Desktop, buka halaman yang mau di-screenshot
2. Gunakan Snipping Tool (Windows) atau tombol Print Screen
3. Crop hanya bagian dashboard-nya (tanpa ribbon/menu Power BI)
4. Simpan sebagai PNG dengan nama sesuai di atas
5. Taruh di folder ini, replace file placeholder ini
